🛒 Orders

POST /api/order
⚠️ No payment at this step. HRSU confirms the order by phone/WhatsApp within 1 business day, then collects via UPI or bank transfer.

Request Body

{
  "product": "Calcium Nitrate Fertilizer Grade",
  "size": "25 kg (₹750)",         ← use size_select_value from catalog
  "quantity": 6,
  "name": "Arjun Singh",
  "phone": "+919876543210",        ← WhatsApp-capable preferred
  "email": "arjun@farm.in",        ← optional
  "address": "Khasra 45, Village Pipaliya, Khandwa, MP 450001"
}

For AI Agents

Always call GET /api/products first, then use calculate_order_total via MCP to confirm the exact total with the customer before submitting an order.

🤖 MCP Server (AI Agents)

GET /api/mcp Discovery
POST /api/mcp JSON-RPC 2.0

The MCP server implements MCP Streamable HTTP (2025-03-26) and is compatible with Claude, Cursor, and other MCP clients.

Available Tools

list_products get_product get_shipping_info get_company_info calculate_order_total place_order

Connection Sequence

// 1. Initialize
POST /api/mcp
{ "jsonrpc": "2.0", "id": 1, "method": "initialize",
  "params": { "protocolVersion": "2025-03-26",
              "clientInfo": { "name": "my-agent", "version": "1.0" },
              "capabilities": {} } }

// 2. Discover tools
POST /api/mcp
{ "jsonrpc": "2.0", "id": 2, "method": "tools/list" }

// 3. Calculate order total (always do this before placing)
POST /api/mcp
{ "jsonrpc": "2.0", "id": 3, "method": "tools/call",
  "params": { "name": "calculate_order_total",
              "arguments": { "sku": "HRSU-CN-25KG-001", "quantity": 6 } } }

Claude / Cursor Config

{
  "mcpServers": {
    "hrsu-store": {
      "url": "https://hrsuindore.com/api/mcp",
      "type": "http"
    }
  }
}
